# Tech Haven
This is a marketplace project. On this site, leads can sell and buy electronic devices. They can also leave their product ratings and comments on it.

This marketplace was developed with the use of:
1. HTML5
2. CSS3
3. JavaScript
4. React.js
5. Vite
6. Vitest
7. REST
8. Redux Toolkit
9. React Router
10. Git
